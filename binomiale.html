2. La Loi Binomiale ℬ(n, p)

Elle modélise le nombre de succès dans une répétition d'épreuves identiques et indépendantes.

Conditions d'application :

  • Une épreuve à 2 issues (Succès / Échec).
  • Répétition n fois de façon indépendante.
  • Probabilité du succès p constante.

Formules à retenir

  • Espérance : $E(X) = n \times p$ (Moyenne sur le long terme)
  • Écart-type : $\sigma(X) = \sqrt{n \times p \times (1-p)}$
Sur calculatrice :
TI : binomFdp(n, p, k)
Casio : BinomialPD(k, n, p)
